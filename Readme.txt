=====|  Как обновить прокси  |=====

В файл `proxy.txt` записать список всех прокси в формате <HOST>:<PORT>:<LOGIN>:<PASS> (1 Прокси на одной строке). Запустить `START_WT.bat`

После запуска создастся файл `proxies.txt` с прокси в  формате http://<LOGIN>:<PASS>@<HOST>:<PORT>. `proxy.txt` переименуется в `proxy.txt.bak`

Если `proxy.txt.bak` уже есть, то нужно удалить `proxies.txt`, переименовать `proxy.txt.bak` в `proxy.txt` и запустить `START_WT.bat`

=====|  Как пользоваться  |=====
1 ПУНКТ - Собирает все объявления в выбранном регионе и городе.
Создается папка `data -> [Название региона]_[ID региона] -> [Название города]_[ID Города]`.
В папке для КАЖДОЙ категории в выбранном городе создается файл в формате `[ID региона]_[ID Города]_[ID Категории]_[Название категории]__offers([Количество объявлений в файле]).xlsx`.
В файле будут собранные объявления.

После завершения собра объявлений, создается файл в папке `data` в формате `merged_[Название региона]_[ID Региона]__[Название города]_[ID Города].xlsx`.
В merged файле сохранены объявления из `data -> [Название региона]_[ID региона] -> [Название города]_[ID Города]`, объединенные в одном файле, при этом удаляются дубликаты по ID Объявления

=====
2 ПУНКТ - Сбор номеров для выбранного merged файла.
Открывается megded файл и проходит по каждой ячейке в столбце С, если в ячейке было значение 'true', то для нее пытается получить номер и записывается в эту ячейку иначе записывается 'не указан'. 
Если произошла ошибка, то в ячейке будет 'Ошибка' или 'Captcha'.
После завершения в названии файла появится знак +.
Можно повторно выбирать этот же файл, тогда ТОЛЬКО ячейки с 'Captcha' обработаются повторно и в них запишется номер если ошибкок не будет. При этом добавится еще один знак +, если он уже был.


=====
3 ПУНКТ - Сбор номеров из файла для уже собранных объявлений.
В папке `data -> [Название региона]_[ID региона] -> [Название города]_[ID Города]` открывается КАЖДЫЙ xslx файл и обрабатывается отдельно аналогично как в пункте 2.


=====
4 ПУНКТ - Объединение файлов уже собранных объявлений и с полученными номерами телефона.
Объединяются файлы из папки `data -> [Название региона]_[ID региона] -> [Название города]_[ID Города]` в один файл и сохраняется в папке `data` в формате `force_merged_[Название региона]_[ID Региона]__[Название города]_[ID Города].xlsx`
